Student Management System
This project is a simple Student Management System built using React and JSON Server. It allows users to view, add, edit, and delete student records.

Features

View a list of students with their details (ID, Name, Grade, Major).
Add new students with their details.
Edit existing student details.
Delete students from the system.


Setup

Clone the repository:

git clone https://github.com/your-username/student-management-system.git


Install dependencies:

Navigate into the project directory and install the necessary dependencies:

cd student-management-system
npm install

Start JSON Server:

Start the JSON Server, which will serve as the backend for managing student data:

npm run server


Start the React application:

Open a new terminal window and start the React application:

npm start


Access the application:

Open your web browser and navigate to http://localhost:3000 to access the Student Management System.

Usage

View the list of students and their details on the homepage.
Use the "Add New Student" button to add a new student.
Click on the "Edit" button next to a student's details to modify their information.
Click on the "Delete" button to remove a student from the system.


Technologies Used

React
JSON Server
HTML
CSS (with Tailwind CSS for styling)
